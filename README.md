## 油管下载器

## Description

​	本项目用于批量从数据库获取待下载链接，并采用 `yt-dlp` 工具下载油管视频到本地，并上传云端(OBS or COS)



## Usage

1. 安装第三方依赖包

   ```
   pip install -r requirements.txt
   ```

2. 创建 `.env` 配置文件在根目录

3. 运行程序

4. （施工中...）
